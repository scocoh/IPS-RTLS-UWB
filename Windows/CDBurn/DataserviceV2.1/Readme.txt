If you need to upgrade to the version 2.1, copy all the files except the web.config into your dataservice directory. Version 2.1 contains a New Zone Report page and a second dataservice for the 8/31/2006 Map table upgrade. Applications using version 2.0 of the ParcoRTLS.dll assembly will still be able use the DataV1.asmx service. The new version 2.1 of the ParcoRTLS.dll must use the DataV2.1.asmx service.
